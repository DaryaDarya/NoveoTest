Rest Api "Корзина магазина"


Конфигурация сервера и базы данных хранится по адресу config/config.json

Следующие урлы доступны для запросов: 
"/api/cart/:product_id/:quantity" - добавляет продукт в корзину
"/api/cart/:product_id" - удаляет продукт из корзины
"/api/cart" - отображает содержимое корзины
"/api/products" - отображает список доступных товаров

Пример заполненной базы в файле "products.json"
